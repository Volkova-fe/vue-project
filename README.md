## Локальная установка и запуск
```
Клонировать репозиторий
  git clone git@github.com:Volkova-fe/vue-project.git

Войти в проект через консоль
  cd mesto-project

Установить зависимости
  npm install

Старт сервера
  npm run dev
  
Адрес сервера

http://127.0.0.1:5173
```
